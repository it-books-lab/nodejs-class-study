# timer

- `setTimeout`와 같은 타이머 함수를 호출하면 즉시 타이머가 등록되고, 지정한 시간 후에 콜백 함수가 실행된다.
- `timeout`에는 타이머의 ID 저장된다.

```jsx
// setTimeout을 호출하면 즉시 타이머가 등록되고, 지정한 시간 후에 콜백 함수가 실행됨.
const timeout = setTimeout(() => { // timeout에는 타이머의 ID 저장.
  console.log('1.5초 후 실행');
}, 1500);

const interval = setInterval(() => { // setInterval은 0초부터 시작이 아니라 n초부터 시작!
  console.log('1초마다 실행');
}, 1000);

const timeout2 = setTimeout(() => {
  console.log('실행되지 않습니다');
}, 3000);

setTimeout(() => {
  clearTimeout(timeout2);
  clearInterval(interval);
}, 2500);

const immediate = setImmediate(() => {
  console.log('즉시 실행');
});

const immediate2 = setImmediate(() => {
  console.log('실행되지 않습니다');
});

clearImmediate(immediate2);
```

```jsx
// 실행 결과
즉시 실행 (immediate)
1초마다 실행 (1초)
1.5초 후 실행
1초마다 실행 (2초)
```

# fs

```jsx
const fs = require('fs');

fs.readFile('./readme.txt', (err, data) => {
  if (err) {
    throw err;
  }
  console.log(data);
  console.log(data.toString());
});
```

- 주의: 읽을 파일의 경로는 현재 파일 기준이 아니라 node 명령어를 실행하는 콘솔 기준이다!
